# ⚡️ 수시 최저 필터 (Susi Passfinder)

### 👉 서비스 바로가기: [https://misslee041022-blip.github.io/susi-filter/]

**"복잡한 모집요강, 클릭 한 번으로 필터링하다"**

**수시 최저 필터**는 수험생들이 자신의 모의고사 등급을 입력하면, 최저학력기준을 충족하는 대학과 학과를 즉시 찾아주는 웹 서비스입니다.

## 🚀 주요 기능
* **개인정보 보호 (Serverless):** 서버 없이 브라우저에서만 작동하여, 성적 정보가 외부로 전송되지 않습니다.
* **엑셀 호환성:** 학교 선생님이나 입시 전문가가 정리한 엑셀 파일(.xlsx)을 그대로 업로드하여 분석할 수 있습니다.
* **동적 필터링:** 학교, 계열, 전형, 최저 유무 등 다양한 조건으로 실시간 필터링이 가능합니다.
* **찜하기 기능:** 관심 있는 학과를 찜해두면(Local Storage), 브라우저를 종료해도 목록이 유지됩니다.
* **샘플 데이터 제공:** 엑셀 파일이 없어도 즉시 체험해볼 수 있는 시연 모드를 지원합니다.

## 🛠️ 기술 스택
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS)
* **Library:** SheetJS (xlsx) - 엑셀 파일 파싱
* **Storage:** LocalStorage - 사용자 찜 목록 저장

## 💻 사용 방법
1. 위 링크를 통해 웹사이트에 접속합니다.
2. 가지고 있는 입시 데이터 엑셀 파일을 업로드하거나, '샘플 데이터로 체험하기' 버튼을 누릅니다.
3. 자신의 국/수/영/탐/한국사 등급을 입력하고 '분석 시작'을 누릅니다.
4. 필터링된 결과를 확인하고 관심 있는 학과를 찜(★)합니다.
